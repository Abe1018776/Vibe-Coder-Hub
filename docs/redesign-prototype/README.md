# Redesign prototype (visual reference)

These files are the owner's static React-on-CDN prototype that defines the new
YidVibe brand and look. They are the **visual source of truth** for the redesign
(see [`../superpowers/specs/2026-05-31-yidvibe-redesign-design.md`](../superpowers/specs/2026-05-31-yidvibe-redesign-design.md)).

- `styles.css` — design tokens, base, nav, buttons, chips, footer, reveal
- `pages.css` — hero, sections, cards, showcase/submit/gigs/etc. component styles

The prototype's **sample data is demo only** — do not treat its projects,
builders, gigs, etc. as real content. The real app is wired to Supabase.

Fonts: **Comfortaa** (display) + **Nunito Sans** (body).
Full prototype (all `.jsx` page files) lives in the owner's original zip; the two
CSS files here carry the reusable design system that the Next.js app ports.
